# Foodie

A food experiment for Ruby on Rails workshop

## Overall features

1. Landing page
2. Featured plates
3. Plate details (with ingredients)
4. Vote system (per ingredient)

## Extra Features

1. Pretty UI (bootstrap)
2. Image Uploads (S3)
3. Authentication (Devise)
4. Heroku
